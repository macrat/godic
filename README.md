godic
=====

goo国語辞典を見るためのコンソールアプリ。

スクレイピングしているだけなので、使用はほどほどに。
突然使えなくなることもあるかもしれません。

## インストール
``` bash
	$ go get github.com/macrat/godic
```

## 使い方
以下のようにして使用すると、マッチした単語に関する説明が表示されます。
``` bash
	$ godic 調べたい単語
```

オプション等は今のところありません。
